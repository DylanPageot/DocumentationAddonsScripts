# Install the Discord Bot

This page allows you to install the Madonn'Admin Discord bot on the Discord server of your choice. Once connected, the bot extends the platform's capabilities directly into your Discord — giving your team access to player information, server status, and more without leaving Discord.

***

## 🛠️ Installation

Installing the bot is straightforward:

1. Click the **Login with Discord** button on the page
2. **Select the Discord server** on which you want to install the Madonn'Admin bot
3. Follow the authorization steps presented by Discord
4. Madonn'Admin takes care of the rest — the bot will be automatically configured for your community

**That's it!** The Discord bot is now ready to use on your server. 🎉
